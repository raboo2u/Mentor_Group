Git Notes

Adding a file into local git repository

Add a new file on your computer (ie, index.html)

git status (Should see file name just added in red)

git commit -m "add a note about what you did"

git status (Should see file name just added in green)

git git push (Pushes the change to github)

git git status (Should see branch is up to date)

git status shows red Changes are not staged

git status shows green Changes are staged ready to commit
